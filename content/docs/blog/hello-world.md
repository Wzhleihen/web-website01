## 为什么选 Hugo

速度快，Markdown 写作，Git 管理，Cloudflare Pages 免费部署——这四点让我选择了 Hugo 而不是 WordPress 或 Hexo。

## 搭建过程

整个搭建过程不超过一小时：

1. `hugo new site .` 初始化
2. Hugo Module 方式引入 Lotus Docs 主题
3. 配置 `hugo.toml`
4. 接入 Cloudflare Pages 自动部署
