## 欢迎

这是基于 [Lotus Docs](https://lotusdocs.dev) 主题搭建的个人技术站点。

## 内容结构

| 栏目 | 内容 |
|---|---|
| **文档** | 技术笔记、学习整理、项目文档 |
| **博客** | 深度技术文章、踩坑经验 |
| **关于** | 个人介绍与联系方式 |

## 技术栈

- **Hugo** v0.162（Extended）
- **Lotus Docs** 主题
- **Cloudflare Pages** 自动部署

## 本地开发

```bash
hugo server -D
# 访问 http://localhost:1313/docs/
```
